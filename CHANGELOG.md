# 更新日志

## v2.0 - Excel伪装版 (2026-06-13)

### 🎨 界面改造
- **完全重写UI** - 从便签样式改为Excel电子表格样式
- **窗口标题** - "Book1 - Excel"，完美伪装
- **表格视图** - 7列专业数据展示，模仿Excel风格
- **颜色主题** - 白底黑字，网格线，交替行颜色

### 📈 新增功能
- **走势图表** - 新增"走势分析"标签页，实时显示价格趋势
- **双视图模式** - 可在数据表格和走势图之间切换
- **红涨绿跌** - 自动颜色标识，上涨红色，下跌绿色
- **完整数据** - 显示代码、名称、最新价、涨跌额、涨跌幅、成交量、最高/最低

### 🔧 技术改进
- 使用 `QTableWidget` 替代自定义组件
- 实现 `ChartView` 图表绘制组件
- 优化数据缓存机制 (`self.stock_data`)
- 改进窗口置顶策略（保留置顶但更自然）

### 📦 依赖更新
- 新增 `numpy>=1.21.0`（为未来数据分析做准备）

### 🎯 隐蔽性提升
- 从"工作笔记"升级为"Excel表格"
- 看起来更像在办公
- 即使被同事看到也不会引起怀疑
