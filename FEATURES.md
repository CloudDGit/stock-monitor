# 新功能使用说明

## v2.1 新增功能

### 1. 智能股票代码识别 

**功能说明：**
- 输入6位股票代码后，系统自动从新浪API获取股票名称
- 无需手动输入名称，提高效率
- 采用后台异步线程，不会阻塞界面

**使用方法：**
1. 点击"+ 添加"按钮
2. 在"代码"输入框中输入6位代码（如：600519）
3. 等待1-2秒，"名称"栏会自动填充
4. 如果显示"✓ 已获取"，点击"确定"即可
5. 如果显示" 无法获取"，可以手动输入名称

**支持的市场：**
- 上海证券交易所（6开头）
- 深圳证券交易所（0、3开头）

### 2. 持仓数据导入

#### 方式一：国泰君安证券API导入

**重要提示：**
国泰君安证券通常不直接开放API给个人用户。要使用此功能需要：
1. 联系国泰君安机构业务部申请API访问权限
2. 签署相关协议并获得API密钥
3. 安装国泰君安提供的SDK

**使用步骤（已获得API权限的情况下）：**
1. 右键点击系统托盘图标
2. 选择"导入持仓" → "国泰君安持仓"
3. 输入资金账号和交易密码
4. 点击"确定"登录
5. 系统自动获取持仓并添加到监控列表

**注意事项：**
- 当前版本仅提供API框架示例
- 实际需要国泰君安官方支持
- 如果无法获取API权限，请使用CSV导入方式

---

#### 方式二：CSV文件导入（推荐）✨

**功能说明：**
- 从券商导出的持仓CSV文件批量导入
- 通用方案，适用于所有券商
- 自动识别常见列名格式

**CSV文件格式要求：**

支持的列名（任意组合）：
- 股票代码列：`代码`、`证券代码`、`股票代码`、`code`、`symbol`
- 股票名称列：`名称`、`证券名称`、`股票名称`、`name`

**示例CSV内容：**
```csv
代码,名称,数量,成本价,当前价,市值,盈亏
600519,贵州茅台,100,1800.00,1850.00,185000.00,5000.00
000858,五粮液,200,150.00,155.00,31000.00,1000.00
```

**使用步骤：**
1. 从券商交易系统导出持仓数据为CSV或Excel格式
2. 如果是Excel文件，另存为CSV格式
3. 右键点击系统托盘图标
4. 选择"导入持仓" → "从CSV文件导入"
5. 选择CSV文件
6. 系统自动解析并添加到监控列表

**示例文件：**
项目根目录下提供了 `example_positions.csv` 示例文件，可以参考其格式。

## 常见问题

### Q1: 为什么输入代码后没有自动获取名称？
A: 可能的原因：
- 网络连接问题，检查网络是否正常
- 股票代码错误，确认是6位有效代码
- API暂时不可用，可以手动输入名称

### Q2: 国泰君安API无法连接怎么办？
A: 这是因为国泰君安未开放个人API权限。解决方案：
- 使用CSV文件导入方式（推荐）
- 联系国泰君安申请机构API权限

### Q3: CSV导入后部分股票没显示？
A: 可能原因：
- CSV格式不符合要求，检查列名是否正确
- 股票代码格式错误，确保是6位数字
- 重复的股票不会重复添加

### Q4: 如何查看已导入的持仓详情？
A: 当前版本仅导入股票代码和名称到监控列表，持仓数量、成本等信息需要自行记录。未来版本会考虑增加持仓管理功能。
