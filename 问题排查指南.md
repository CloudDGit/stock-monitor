# 问题排查指南 v3.0

## 已知修复的问题

### ✅ 已修复

1. **最小化后托盘图标不显示**
   - 原因：窗口标志设置不当
   - 修复：改用标准窗口标志 `Qt.Window`
   
2. **双击股票无反应**
   - 原因：ExcelStyleTable未正确获取主窗口引用
   - 修复：使用`self.window()`获取主窗口，并添加空值检查

3. **图表标题更新失败**
   - 原因：使用findChild查找QLabel不可靠
   - 修复：直接使用`self.chart_widget.title_label`属性

---

## 可能存在的问题

### 1. 表格显示问题

**现象：** 表格列数不对或字段显示错误

**检查方法：**
- 打开程序后查看是否有8列
- 列标题应为：名称、代码、市值、盈亏、盈亏%、当日盈亏、当日%、成本/现价

**可能原因：**
- ExcelStyleTable初始化时setColumnCount(8)未生效
- refresh_table中列索引错误

**解决方法：**
```python
# 在init_ui中确认
self.setColumnCount(8)

# 在refresh_table中确认列索引0-7都有数据
```

---

### 2. 持仓盈亏显示为0

**现象：** 所有股票的盈亏都显示0或"--"

**可能原因：**
- positions.json文件不存在或格式错误
- 股票代码不匹配

**检查方法：**
1. 查看`positions.json`文件是否存在
2. 确认股票代码与stocks.json中的代码一致
3. 检查控制台是否有加载错误

**解决方法：**
```bash
# 删除positions.json，让程序重新创建
del positions.json
# 重启程序
python stock_monitor.py
```

---

### 3. 底部汇总栏不显示

**现象：** 底部状态栏没有显示总市值、总盈亏等信息

**可能原因：**
- update_total_summary方法未被调用
- status_label未正确创建

**检查方法：**
- 查看refresh_table末尾是否调用了update_total_summary
- 确认status_label在init_ui中已创建

---

### 4. 分时图不显示

**现象：** 双击股票后切换到走势图标签页，但图表为空

**可能原因：**
- 没有实时数据（非交易时间）
- ChartView绘制逻辑有误
- minute_data为空

**当前状态：**
- v3.0版本的分时图功能尚未完全实现
- fetch_and_draw_minute_chart方法目前是空的（pass）

**临时方案：**
- 使用简单价格趋势图（update_price方法）
- 等待后续版本完善分时数据获取

---

### 5. 数据刷新慢或无数据

**现象：** 股票数据长时间不更新或显示"--"

**可能原因：**
- 网络问题导致无法连接新浪财经API
- StockDataFetcher线程异常退出
- API返回数据格式变化

**检查方法：**
1. 查看控制台是否有错误信息
2. 检查网络连接
3. 手动访问 http://hq.sinajs.cn/?t=xxx&list=sh600519 测试API

**解决方法：**
- 点击"刷新"按钮手动刷新
- 检查防火墙是否阻止Python访问网络
- 查看last_update_label确认最后更新时间

---

### 6. 添加股票时自动获取名称失败

**现象：** 输入代码后名称栏不自动填充

**可能原因：**
- 后台线程异常
- API请求超时
- 股票代码无效

**检查方法：**
- 查看fetch_status_label的提示文字
- 确认输入的是6位有效代码

**解决方法：**
- 稍等几秒（异步获取需要时间）
- 如果显示"✗ 无法获取"，可手动输入名称
- 检查网络连接

## 调试技巧

### 1. 查看控制台输出

运行程序时保留命令行窗口，查看是否有错误信息：
```bash
python stock_monitor.py
```

### 2. 启用详细日志

在关键位置添加print语句：
```python
print(f"DEBUG: 加载了 {len(self.stocks)} 只股票")
print(f"DEBUG: 持仓数据: {self.positions}")
```

### 3. 检查数据文件

**stocks.json** - 应该包含股票代码和名称列表
```json
[
  ["688012", "中微公司"],
  ...
]
```

**positions.json** - 应该包含持仓详细信息
```json
{
  "688012": {
    "name": "中微公司",
    "quantity": 200,
    "cost_price": 277.088
  },
  ...
}
```

### 4. 使用Python调试器

```bash
python -m pdb stock_monitor.py
```

## 快速重置

如果程序出现严重问题，可以快速重置：

```bash
# 1. 备份重要数据（如果需要）
copy stocks.json stocks.json.bak
copy positions.json positions.json.bak

# 2. 删除配置文件
del stocks.json
del positions.json

# 3. 重启程序（会自动创建默认配置）
python stock_monitor.py
```
