# StockVision - 智能股票监控与分析系统

![Python](https://img.shields.io/badge/Python-3.7+-blue.svg)
![PyQt5](https://img.shields.io/badge/PyQt5-GUI-green.svg)
![License](https://img.shields.io/badge/License-MIT-orange.svg)

一款专业的 A 股实时监控与分析工具，采用 Excel 表格样式设计，支持持仓盈亏精准计算、实时分时图显示、自动化交易记录等功能。

---

## 核心特性

### 1. 实时行情监控
- **多源数据获取**：集成新浪财经、腾讯财经多数据源，自动容错切换
- **10秒极速刷新**：实时掌握股价变动
- **智能股票识别**：输入代码自动获取股票名称

### 2. 精准盈亏计算
- **总盈亏**：基于加权平均成本价实时计算
- **当日盈亏**：基于昨收价计算当日浮动盈亏
- **追加买入处理**：自动区分原有持仓与当日买入部分，分别计算盈亏
- **卖出收益计算**：含印花税、佣金等费用的实际收益计算
- **当日新开/增买标记**：智能识别并正确处理当日交易

### 3. 可视化分析
- **专业分时图**：仿同花顺风格，支持十字光标精确读数
- **迷你分时图**：表格内嵌微缩走势图，一目了然
- **实时统计面板**：市值、盈亏等多维度汇总展示

### 4. 数据管理
- **多格式导入**：支持 CSV、XLS、XLSX、TSV 等券商导出格式
- **券商直连**：支持国泰君安等主流券商账号导入
- **自动同步**：一键刷新最新持仓数据
- **持久化存储**：JSON 格式本地存储，程序重启不丢失

### 5. 交易管理
- **买入交易**：自动计算费用、调整持仓成本
- **卖出交易**：支持全部/部分卖出，实时更新盈亏
- **交易记录**：自动标记当日交易类型

## 快速开始

### 环境要求
- Python 3.7+
- Windows/macOS/Linux

### 安装

```bash
# 克隆项目
git clone https://github.com/CloudDGit/stock-monitor.git
cd stock-monitor

# 安装依赖
pip install -r requirements.txt
```

### 运行

```bash
python stock_monitor.py
```

或双击 `start.bat`（Windows）

## 功能详解

### 持仓盈亏计算规则

| 场景 | 总盈亏计算 | 当日盈亏计算 |
|------|----------|------------|
| 普通持仓 | (现价 - 成本价) × 数量 | (现价 - 昨收价) × 数量 |
| 当日新开 | (现价 - 买入价) × 数量 | = 总盈亏 |
| 追加买入 | (现价 - 加权成本) × 总数量 | 原有持仓按昨收价 + 当日买入按买入价 |

### 费用计算

**买入费用：**
- 佣金：成交金额 × 0.025%（最低5元）
- 过户费：成交金额 × 0.02‰（沪市）

**卖出费用：**
- 印花税：成交金额 × 0.1%（仅卖出收取）
- 佣金：成交金额 × 0.025%（最低5元）
- 过户费：成交金额 × 0.02‰（沪市）
