# A股实时监控工具 - 隐蔽版 v3.0

一款专为上班族设计的隐蔽A股监控工具，伪装成Excel电子表格样式，极难被同事发现。支持持仓盈亏实时监控和分时图显示。

## ✨ 新功能 (v3.0)

- **💼 持仓盈亏监控** - 实时计算并显示总市值、总盈亏、当日盈亏
- **📊 智能表格布局** - 按照券商APP风格调整字段：名称、代码、市值、盈亏、盈亏%、当日盈亏、当日%、成本/现价
- **⏱️ 分时图显示** - 双击股票查看分时走势图，自动识别交易日
- **📈 底部汇总栏** - 实时显示账户总盈亏情况
- **🔧 修复托盘问题** - 最小化后托盘图标正常显示
- ** 预置测试数据** - 内置11只真实持仓股票数据

### v2.1 功能回顾
- **🔍 智能代码识别** - 输入股票代码后自动获取股票名称
- **📥 持仓数据导入** - 支持从券商系统或CSV文件导入个人持仓

## 功能特点

- **极致隐蔽**: 从外观到交互完全模仿Excel，即使被看到也不会引起怀疑
- **实时数据**: 自动从新浪财经获取实时股票数据，10秒刷新一次
- **多股监控**: 可同时监控多只关注的股票
- **持仓管理**: 记录持仓成本和数量，实时计算盈亏
- **双视图模式**: 
  - "数据监控"标签页 - Excel表格视图（8列详细数据）
  - "走势分析"标签页 - 分时/趋势图表
- **系统托盘**: 可最小化到系统托盘，后台运行
- **双击交互**: 双击股票自动切换到走势图

## 使用方法

1. 运行程序：
```bash
python stock_monitor.py
```
或双击 `start.bat`

2. **查看持仓盈亏**：
   - 程序启动后自动加载预设的11只持仓股票
   - 表格显示：名称、代码、市值、盈亏、盈亏%、当日盈亏、当日%、成本/现价
   - 底部状态栏显示总市值、总盈亏、今日盈亏汇总

3. **查看分时图**：
   - 双击任意股票行
   - 自动切换到"走势分析"标签页
   - 显示该股票的分时走势图

4. **添加监控股票**：
   - 点击"+ 添加"按钮
   - 输入股票代码（如：600519）
   - **系统自动获取股票名称** ✨
   - 如果无法自动获取，可手动输入

5. **导入个人持仓**（可选）：
   - **右键点击系统托盘图标**
   - 选择"导入持仓" → "国泰君安持仓" 或 "从CSV文件导入"
   - 登录后自动将持仓股票添加到监控列表

## 界面说明

### 数据监控标签页（8列布局）

| 列名 | 说明 | 颜色规则 |
|------|------|----------|
| 名称 | 股票简称 | - |
| 代码 | 6位代码 | 灰色 |
| 市值 | 持仓数量 × 当前价 | 黑色加粗 |
| 盈亏 | (当前价 - 成本价) × 数量 | 红涨绿跌 |
| 盈亏% | 盈亏 / 成本 × 100% | 红涨绿跌 |
| 当日盈亏 | 涨跌额 × 数量 | 红涨绿跌 |
| 当日% | 当日涨跌幅 | 红涨绿跌 |
| 成本/现价 | 成本价/当前价 | - |

### 底部汇总栏
实时显示三项关键数据：
- **总市值** - 所有持仓股票的当前市值总和
- **总盈亏** - 累计盈亏金额（红色为正，绿色为负）
- **今日盈亏** - 当日盈亏金额（红色为正，绿色为负）

### 走势分析标签页
- 实时价格趋势图
- 最近100个数据点
- 自动缩放适应价格范围
- 显示最高价、最低价、成交量

## 常见问题

### Q1: 为什么有些股票的盈亏显示为0？
A: 需要在 `positions.json` 文件中设置该股票的持仓数量和成本价。如果没有设置，系统会认为你没有持仓。

### Q2: 如何修改持仓成本？
A: 直接编辑 `positions.json` 文件，修改对应股票的 `cost_price` 和 `quantity` 字段。

### Q3: 分时图为什么不显示？
A: 分时图需要交易日的实时数据。如果是非交易日或收盘后，可能没有数据。当前版本使用模拟数据演示。

### Q4: 最小化后找不到程序？
A: v3.0已修复此问题。现在最小化后会在系统托盘显示图标，双击即可恢复。
